# 🥛 YourDairyBro

A Spring Boot based Dairy Management Web Application.

## 🚀 Features
- User Login & Registration
- Dairy product management
- Database integration using MySQL
- MVC architecture

## 🛠 Tech Stack
- Java
- Spring Boot
- MySQL
- Maven
- HTML/CSS

## ▶ How to Run
1. Clone the repository
2. Open in STS or VS Code
3. Run the main Spring Boot application class
4. Open browser at http://localhost:8080
